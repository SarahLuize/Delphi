 ## Nota para mim mesma

| Delphi IDE | Lazarus IDE |
| :---: | :---: |
| .DPROJ | .LPI |
| .DFM | .LFM |

Form é composto pelo arquivo `.DFM` (ou `.LFM` dependendo da IDE) + arquivo `.PAS`  
`.DFM` / `.LFM` é a parte visual e `.PAS` é a codificação.

| Extensão | Função no Lazarus IDE | Equivalente Delphi IDE |
| :---: | :---: | :---: |
| .pas | Código-fonte das unidades | .pas |
| .lfm | Formulário, layout e propriedades dos componentes | .dfm |
| .lpr | Programa principal, ponto de entrada | .dpr |
| .lpi | Configurações do projeto (arquivos, opções) | .dproj |
| .lps | Backup do .lpi | ... |
| .res | Recursos do projeto (ícones, imagens) | .res |
| .ico | Ícone do projeto | .ico |
